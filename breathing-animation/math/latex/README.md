# LaTeX Breathing Cheat Sheet

This directory contains a LaTeX source file to generate a printable "Cheat Sheet" of breathing patterns using TikZ for visualization.

## Usage

1.  Make sure you have a LaTeX distribution installed (e.g., `texlive`, `mactex`).
2.  Compile the PDF:
    ```bash
    pdflatex breathing.tex
    ```
3.  Open `breathing.pdf` to view the printable visual guide.
