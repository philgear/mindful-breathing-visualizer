# Swift (iOS) Breathing Visualizer

This directory contains a SwiftUI implementation of the breathing animation, featuring **Native Audio** and **Shape Customization**.

## Features

-   **Three Shapes**: Circle, Square, Lotus (custom Paths).
-   **Native Audio**: Real-time synthesis using `AVFoundation`, `AVAudioEngine`, and `AVAudioSourceNode`.
-   **Architecture**: Safe concurrency with Swift Actors (`@MainActor`).
-   **Design**: Uses the **Serene Palette** (Emerald/Blue/Rose).

## Usage

1. Open Xcode and create a new **iOS App** project.
2. Replace `ContentView.swift` with the file in this directory.
3. Run the simulator.

## Requirements

- iOS 14.0+
- Xcode 12+
